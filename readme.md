# Codeigniter Starterkit

## Installation
 1. `composer install`.
 2. list command run `php ci`

## Cheatsheet

### Code Style
in Windows, run `vendor\bin\phpcs --standard=PSR2 application/controllers` for code style.

### Dump
You can use `dd($foo)` for better var_dump.
